NIMBLE

I used to work at a small company and we wrote off thousands of dollars each month because product was not tracked properly. We could have benefitted greatly from some functionality that an ERP system brings. But, being a small business we could not afford, and didn’t need the robustness of an ERP system.

This app gives the user the ability to keep track of inventory quantity and location within the warehouse. Any user can scan an inventory item’s barcode which allows the user to edit the inventory item or add the item to inventory if it doesn’t already exist. The user also has the ability to add alerts for specific inventory items. An example of the alert feature would be if an item is damaged, then a text alert with the part name and alert message would be sent out to all warehouse staff.

Nimble’s basic functions of tracking inventory, barcode scanning and text alerts add huge value to a small company, at a low cost alternative.

Note: if scanning a barcode, make sure it is a Code-39 barcode. If trying to create alerts, Twilio will only send me the texts because I am using the free version.
